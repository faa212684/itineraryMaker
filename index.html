Welcome to Plan My Day

Tired of planning your own trips? Fear no more! This final project plans everything you need and more! We have created a website to randomly plan a day at any given location (works best in big cities). This includes planning three meals, breakfast, lunch, and dinner, along with two nearby attractions, producing a detailed report.







Note: Try not to spam. The API will stop working after a certain number of requests each day, meaning the website will be rendered unusable
